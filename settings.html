SwarmUI Integration Settings

This extension connects to a running SwarmUI instance to generate images.

  • SwarmUI URL: The address of your SwarmUI web interface. Usually http://127.0.0.1:7804.
  • Show Generation Button: Toggles the image generation button in the chat input area.
  • LLM Image Prompt: The template used to ask the current LLM to generate a visual prompt. This is what the AI sees.
  • Append to Existing Prompt: When checked, the new prompt from the LLM is added to whatever you already have in the SwarmUI prompt box. When unchecked, it replaces it completely.
  • Number of Images: How many images to request from SwarmUI at once.
